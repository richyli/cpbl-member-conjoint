中華職棒球團付費會員方案的球迷偏好與付費意願：一個選擇式聯合分析

本版（v05）依擴充後資料（初始 127、有效 N=115）重跑全分析並更新所有數據。本研究重點在於比較不同球團粉絲對各會員權益屬性的偏好差異，故結果章以球隊 × 屬性交互項模型為核心。引言、文獻與討論待後續版本補足；標 [需補文獻] 處待補。

研究方法

以對比選擇任務逼近真實入會決策

本研究採選擇式聯合分析（choice-based conjoint, CBC；McFadden, 1974）量測球迷對球團付費會員方案的偏好。相較於逐項詢問球迷「重視哪些權益」的自陳式量表，對比選擇任務要求受訪者在權益與價格俱不完美的兩個方案之間做出取捨，使受訪者必須以實際決策揭露其權衡結構，較貼近真實的入會抉擇情境（Train, 2009）。

六個屬性涵蓋金錢、功能、內容、情感四個面向

方案以六個屬性描述，水準數依設計需要設為四、四、三、四、三、四，整理如表 1。年度會員費自九百元至兩千九百元分四級，跨距約三點二倍，兩端分別貼近最低與最高配權益的內容價值，以避免最低價方案因「物超所值」被反射性選取、最高價方案因「定價遠超內容」被反射性捨棄。獨家媒體內容、優先購票順位、互動活動、免費票券與會員周邊則涵蓋功能、內容、情感與社群面向；其中優先購票順位固定每場四張，僅變動順位高低，以分離順位與張數兩個變因。各屬性之納入係參酌中華職棒六支球團現行會員方案的權益結構與球迷社群討論的關注焦點而定（屬性理據詳見文獻回顧，待後續版本補足）。

每位受訪者完成八輪對比並附帶品質檢核

每位受訪者於選定支持球隊後，完成八輪正式對比選擇，每輪呈現兩個方案（方案 A 與方案 B）並保留「兩者皆不選」的選項。屬性的呈現順序在受訪者之間隨機、在受訪者之內固定；各方案的屬性水準於每輪即時隨機產生，並以演算法排除完全相同、價格水準相差超過兩級、以及一方在所有屬性上皆不劣於另一方的支配組合，以確保每一輪都構成有意義的權衡。題組中另插入一題注意力檢核題（指示受訪者直接點選指定方案）、一題鏡像重複題（將第三輪左右對調後重現），以及一題與會員偏好理論上無關的標記題（「我很喜歡白色」），分別用以辨識未專注作答者、偏好不一致者，以及估計共同方法變異。

條件羅吉特模型估計部分效用

選擇資料以條件羅吉特模型（conditional logit; McFadden, 1974）估計。方案 $i$ 在選擇集 $J$ 中被選取的機率為

$$\Pr(i \mid J) = \frac{\exp(X_i^\top \beta)}{\sum_{j \in J} \exp(X_j^\top \beta)},$$

其中 $X_i$ 為方案 $i$ 的屬性設計向量，$\beta$ 為待估的部分效用參數。鑑於資料中近兩成的選擇落在「兩者皆不選」，本研究將其視為第三個替代方案，並納入一個「皆不選」特定常數（none alternative-specific constant），以分離方案內在吸引力與屬性效果。各屬性水準採效果編碼（effects coding，sum-to-zero；效果編碼），使截距代表樣本平均偏好、各部分效用為偏離平均之量，最末一個水準的部分效用由其餘水準之負和回推。屬性相對重要性以全距法（range method）計算，即各屬性部分效用全距占所有屬性全距總和之百分比。

以交互項模型檢定球隊間的偏好差異

為回答本研究核心問題（不同球團粉絲對各權益屬性的偏好是否不同），本研究在合併樣本的條件羅吉特模型中加入「球隊 × 屬性水準」交互項，以人數最多的球隊為參考組，其餘各隊與全部屬性的效果編碼變項交互。透過比較「含交互項」與「不含交互項」兩模型的對數概似比檢定（likelihood-ratio test），檢定球隊是否整體調節屬性偏好；再逐屬性移除其交互項並重估，以各屬性交互項的聯合顯著性判定「哪些屬性的偏好在隊間有差異」。此設計借合併樣本保留考驗力，避免將百餘人的樣本硬拆為六個各自估計而導致估計不穩。

以 Python 套件實作估計與信效度檢驗

所有分析以 Python 配合 pandas、numpy、scipy、scikit-learn、statsmodels 與 xlogit 套件實作；原始問卷資料的屬性水準對應與品質篩除，由腳本依水準標記（token）比對推斷。條件羅吉特之對數概似以 L-BFGS-B 最佳化求解；願付金額信賴區間以 Krinsky-Robb 蒙地卡羅模擬（Krinsky & Robb, 1986）由叢集穩健共變異矩陣抽取五千次估得。

樣本與品質篩除

本研究透過自製網頁問卷收集回應，初始樣本為 127 位中華職棒球迷。品質篩除依四項標準執行：標準一為「兩者皆不選」比例達八成以上者（剔除 5 人）；標準二為有效作答題數不足三題者（無）；標準三為以初步模型輔助辨識的系統性選擇劣勢方案者（剔除 1 人）；標準四為事後支配檢核（post-hoc Pareto dominance check）。標準四僅就具明確方向的三個屬性（會員費越低越好、購票順位越前越好、免費票券越多越好）且其餘類別型屬性兩方案相同的「真正支配」題判定，在此保守定義下無受訪者於支配題選取被支配方；另有 6 位受訪者在所有可權衡的對比題中始終選取同一側，判定為非交易者（non-trader）而剔除。最終有效樣本為 115 人，分屬六支球團（中信兄弟 32、統一獅 21、富邦悍將 20、樂天桃猿 20、味全龍 14、台鋼雄鷹 8）。

樣本規模之說明

受訪者規模在聯合分析的經驗法則下屬中型（Orme, 2010 建議的大樣本門檻）；本研究於後文穩健性與限制一節據實揭露其對統計考驗力的影響，並以分群結構與信效度檢驗佐證主結論的方向穩健性。

研究發現

價格為最關鍵屬性，互動與票券次之

六個屬性的相對重要性與部分效用整理如表 2。年度會員費為最關鍵屬性，相對重要性達 31.5%，明顯高於其餘屬性；互動活動（17.2%）、免費票券（15.9%）與優先購票順位（15.4%）居次，獨家媒體內容（12.2%）與會員周邊（7.8%）影響相對較小。價格的部分效用隨金額單調遞減（九百元 $+0.65$、兩千九百元 $-0.59$），符合理論預期：會費越低，方案效用越高。

球迷對深度內容偏好明確、對見面會抽選反而排斥

獨家媒體內容中，球員深度專訪的部分效用最高（$+0.30$），二軍與新秀養成紀錄最低（$-0.19$），顯示球迷偏好聚焦於一軍明星球員的深度內容。值得注意的是，互動活動呈現非單調且反直覺的型態：「無互動活動」的部分效用最高（$+0.36$），而線上球迷問答（$-0.32$）與啦啦隊見面會抽選（$-0.16$）皆為負值。此一結果意味著，以抽選為機制的見面會權益對多數球迷而言並非加分項，反而因中籤機率低、形式化程度高而被視為負擔。免費票券與優先購票順位則皆隨水準單調遞增，符合「票券越多、順位越前越好」的預期。

會員方案具內在吸引力

「皆不選」特定常數估計為 $-1.06$，為負值，顯示在控制各屬性效果之後，受訪者整體仍傾向選擇加入會員而非皆不選，亦即球團付費會員方案本身具有內在吸引力，而非僅靠個別權益堆疊吸引球迷。

願付金額顯示票券、順位與深度內容最具貨幣價值

以價格部分效用對實際金額迴歸得邊際效用 $\beta_{\text{price}} = -0.000642$（每元），各非價格屬性水準相對其最低水準的願付金額（以邊際替代率換算、Krinsky-Robb 95% 信賴區間）整理如表 3。願付金額顯著為正（信賴區間不含零）者包括：免費票券由零張增至四張（約 $+1{,}114$ 元）、優先購票順位升至第二順位（約 $+1{,}065$ 元）、以及球員深度專訪（約 $+853$ 元）。互動活動的各水準願付金額多為負值，與其負向部分效用一致。將各屬性最佳水準相對最低水準的願付金額加總，堆疊願付金額約為 2,985 元/年，可視為定價的理論參考上限，略高於現行最高會費水準（2,900 元）。

行為與態度雙軌分群揭示三類球迷

為刻劃偏好的個體異質性，本研究並行兩條分群路徑。其一以受訪者的選擇行為特徵（價格敏感度、皆不選比例、方向性屬性勝率）執行 K 平均集群分析（K-means）；其二以球隊認同、球迷社群認同與知覺價值三組共十九題李克特量表執行潛在類別分析（latent class analysis；Latent Class）。兩條路徑均報告三群結果，以兼顧資料驅動的最適解與群數的可解釋性（潛在類別與集群法綜述）。

行為軸區分價格敏感、內容黏著與票券觀望三型

行為分群（K 平均，Silhouette $=0.25$）將 115 位球迷分為三型，命名與規模整理如表 4。價格敏感型（n = 62，54%）價格敏感度偏高、幾乎不選皆不選，為最大群；內容黏著型（n = 32，28%）價格敏感度最低、皆不選比例低，顯示其入會決策由權益內容而非價格主導；票券觀望型（n = 21，18%）皆不選比例最高、對免費票券的勝率最高，反映其對方案整體興趣有限、僅對票券誘因有反應。

態度軸區分三個認同強度層級

態度潛在類別分析將球迷依球隊、社群與價值認同強度分為三個層級：中度認同球迷（n = 37，三量表均值約 4.5 至 4.8）、高度認同球迷（n = 47，約 5.6 至 6.0）、與鐵粉等級的極高認同球迷（n = 31，約 6.0 至 6.2）。兩條分群路徑的交叉表 Cramér's V 僅為 0.13，顯示球迷的「選擇行為型態」與「態度認同強度」近乎獨立：高認同球迷未必對價格不敏感，價格敏感者亦未必認同度低。兩軸因而並列呈現，分別刻劃行為區隔與態度潛在類別。

不同球團粉絲的偏好差異

球隊整體調節屬性偏好

本研究的核心問題在於：不同球團的粉絲，對會員權益屬性的偏好是否不同。球隊 × 屬性交互項模型的概似比檢定顯示，含交互項模型顯著優於不含交互項模型（$\mathrm{LR} = 129.26$, $\mathit{df} = 80$, $p < .001$），亦即球隊整體確實調節了屬性偏好，至少有部分屬性的偏好在球隊之間存在差異。

差異集中於互動活動與免費票券，而非價格與內容

逐屬性的交互項聯合檢定結果整理如表 5。球隊間偏好差異達顯著的屬性為互動活動（$p < .001$）與免費票券（$p < .001$）兩者；年度會員費（$p = .22$）、獨家媒體內容（$p = .29$）、優先購票順位（$p = .23$）與會員周邊（$p = .12$）的偏好則在球隊之間未達顯著差異。換言之，「價格越低越好、偏好球員深度內容」是跨球團粉絲共通的偏好結構，球團之間真正分歧的，是互動活動與免費票券這類情感與社群連結型的權益，其價值高度依附於各球團的應援文化與球迷社群特性。此一結論在樣本由 74 人逐步擴充至 115 人的四次重估中保持一致，顯示其穩健性。

模型配適與信效度檢驗支持主結論

模型在保留任務上的樣本外命中率為 57.8%（高於三選一隨機基準 33.3%），保留集 McFadden pseudo-$R^2$ 為 0.15；以一百次任務軸折半重抽估得的 Spearman-Brown 信度，屬性重要性為 .82、部分效用為 .91；三組態度量表的 Cronbach's $\alpha$ 介於 .85 至 .88；以標記題（「我很喜歡白色」）執行的共同方法變異檢驗顯示，標記題與三個態度量表的相關皆低（最大 $|r| = .12$）、與各屬性個體勝率的相關亦低（最大 $|r| = .12$），且依 Lindell 與 Whitney（2001）的校正法調整後量表間相關幾無實質變動（最大變動 $.024$），顯示共同方法偏誤未構成威脅；資料飽和曲線於現有樣本已大致平穩。混合羅吉特（隨機參數）對條件羅吉特的概似比檢定顯著（$\mathrm{LR} = 86.69$, $\mathit{df} = 16$, $p < .001$），表示存在個體偏好異質性、IIA 假設不足，惟兩模型的係數同號通過十六分之十五（僅一個近於零的係數於重估後變號），且個體層次異質性已由潛在類別分群吸收，故主分析所揭露的偏好方向穩健（IIA 處理與完整檢驗結果詳見附錄 A）。綜合各項指標，命中率、信度、量表效度與共同方法檢驗均達或接近慣例門檻，統計考驗力隨樣本擴充提升至約 .58，整體信效度足以支撐前述偏好結構與球隊差異之推論。
