中華職棒球團付費會員方案的球迷偏好與付費意願：一個選擇式聯合分析

本版（v01）依 /conjoint-run 分析結果撰寫研究方法與研究發現兩章。引言、文獻與討論待後續版本補足；標 [需補文獻] 處待補。

研究方法

以對比選擇任務逼近真實入會決策

本研究採選擇式聯合分析（choice-based conjoint, CBC；McFadden, 1974）量測球迷對球團付費會員方案的偏好。相較於逐項詢問球迷「重視哪些權益」的自陳式量表，對比選擇任務要求受訪者在權益與價格俱不完美的兩個方案之間做出取捨，使受訪者必須以實際決策揭露其權衡結構，較貼近真實的入會抉擇情境（Train, 2009）。

六個屬性涵蓋金錢、功能、內容、情感四個面向

方案以六個屬性描述，水準數依設計需要設為四、四、三、四、三、四，整理如表 1。年度會員費自九百元至兩千九百元分四級，跨距約三點二倍，兩端分別貼近最低與最高配權益的內容價值，以避免最低價方案因「物超所值」被反射性選取、最高價方案因「定價遠超內容」被反射性捨棄。獨家媒體內容、優先購票順位、互動活動、免費票券與會員周邊則涵蓋功能、內容、情感與社群面向；其中優先購票順位固定每場四張，僅變動順位高低，以分離順位與張數兩個變因。各屬性之納入係參酌中華職棒六支球團現行會員方案的權益結構與球迷社群討論的關注焦點而定（屬性理據詳見文獻回顧，待後續版本補足）。

每位受訪者完成八輪對比並附帶品質檢核

每位受訪者於選定支持球隊後，完成八輪正式對比選擇，每輪呈現兩個方案（方案 A 與方案 B）並保留「兩者皆不選」的選項。屬性的呈現順序在受訪者之間隨機、在受訪者之內固定；各方案的屬性水準於每輪即時隨機產生，並以演算法排除完全相同、價格水準相差超過兩級、以及一方在所有屬性上皆不劣於另一方的支配組合，以確保每一輪都構成有意義的權衡。題組中另插入一題注意力檢核題（指示受訪者直接點選指定方案）與一題鏡像重複題（將第三輪左右對調後重現），分別用以辨識未專注作答者與偏好不一致者。

條件羅吉特模型估計部分效用

選擇資料以條件羅吉特模型（conditional logit; McFadden, 1974）估計。方案 $i$ 在選擇集 $J$ 中被選取的機率為

$$\Pr(i \mid J) = \frac{\exp(X_i^\top \beta)}{\sum_{j \in J} \exp(X_j^\top \beta)},$$

其中 $X_i$ 為方案 $i$ 的屬性設計向量，$\beta$ 為待估的部分效用參數。鑑於資料中近兩成的選擇落在「兩者皆不選」，本研究將其視為第三個替代方案，並納入一個「皆不選」特定常數（none alternative-specific constant），以分離方案內在吸引力與屬性效果。各屬性水準採效果編碼（effects coding，sum-to-zero；效果編碼），使截距代表樣本平均偏好、各部分效用為偏離平均之量，最末一個水準的部分效用由其餘水準之負和回推。屬性相對重要性以全距法（range method）計算，即各屬性部分效用全距占所有屬性全距總和之百分比。

以 Python 套件實作估計與信效度檢驗

所有分析以 Python 配合 pandas、numpy、scipy、scikit-learn、statsmodels 與 xlogit 套件實作；原始問卷資料的屬性水準對應與品質篩除，由腳本依水準標記（token）比對推斷。條件羅吉特之對數概似以 L-BFGS-B 最佳化求解；願付金額信賴區間以 Krinsky-Robb 蒙地卡羅模擬（Krinsky & Robb, 1986）由叢集穩健共變異矩陣抽取五千次估得。

樣本與品質篩除

本研究透過自製網頁問卷收集回應，初始樣本為 84 位中華職棒球迷。品質篩除依四項標準執行：標準一為「兩者皆不選」比例達八成以上者（剔除 5 人）；標準二為有效作答題數不足三題者（無）；標準三為以初步模型輔助辨識的系統性選擇劣勢方案者（無）；標準四為事後支配檢核（post-hoc Pareto dominance check）。標準四僅就具明確方向的三個屬性（會員費越低越好、購票順位越前越好、免費票券越多越好）且其餘類別型屬性兩方案相同的「真正支配」題判定，在此保守定義下無受訪者於支配題選取被支配方；另有 5 位受訪者在所有可權衡的對比題中始終選取同一側，判定為非交易者（non-trader）而剔除。最終有效樣本為 74 人。

樣本規模之說明

受訪者規模在聯合分析的經驗法則下屬偏小（Orme, 2010 建議的大樣本門檻）；本研究於後文穩健性與限制一節據實揭露其對統計考驗力的影響，並以分群結構與信效度檢驗佐證主結論的方向穩健性。

研究發現

價格為最關鍵屬性，內容與互動次之

六個屬性的相對重要性與部分效用整理如表 2。年度會員費為最關鍵屬性，相對重要性達 35.3%，遠高於其餘屬性；獨家媒體內容（18.2%）與互動活動（17.0%）次之，免費票券（13.0%）、優先購票順位（9.6%）與會員周邊（6.9%）影響相對較小。價格的部分效用隨金額單調遞減（九百元 $+0.65$、兩千九百元 $-0.65$），符合理論預期：會費越低，方案效用越高。

球迷對深度內容偏好明確、對見面會抽選反而排斥

獨家媒體內容中，球員深度專訪的部分效用最高（$+0.32$），二軍與新秀養成紀錄最低（$-0.35$），顯示球迷偏好聚焦於一軍明星球員的深度內容。值得注意的是，互動活動呈現非單調且反直覺的型態：「無互動活動」的部分效用最高（$+0.38$），而線上球迷問答（$-0.23$）與啦啦隊見面會抽選（$-0.25$）皆為負值。此一結果意味著，以抽選為機制的見面會權益對多數球迷而言並非加分項，反而因中籤機率低、形式化程度高而被視為負擔。免費票券與優先購票順位則皆隨水準單調遞增，符合「票券越多、順位越前越好」的預期。

會員方案具內在吸引力

「皆不選」特定常數估計為 $-1.07$，為負值，顯示在控制各屬性效果之後，受訪者整體仍傾向選擇加入會員而非皆不選，亦即球團付費會員方案本身具有內在吸引力，而非僅靠個別權益堆疊吸引球迷。

願付金額顯示深度內容、票券與順位最具貨幣價值

以價格部分效用對實際金額迴歸得邊際效用 $\beta_{\text{price}} = -0.000674$（每元），各非價格屬性水準相對其最低水準的願付金額（以邊際替代率換算、Krinsky-Robb 95% 信賴區間）整理如表 3。願付金額顯著為正（信賴區間不含零）者包括：免費票券由零張增至四張（約 $+900$ 元）、優先購票順位升至第二順位（約 $+818$ 元）、以及球員深度專訪（約 $+841$ 元）。互動活動的各水準願付金額多為負值，與其負向部分效用一致。將各屬性最佳水準相對最低水準的願付金額加總，堆疊願付金額約為 2,086 元/年，可視為定價的理論參考上限，與實際會費水準（900 至 2,900 元）大致相符。

行為與態度雙軌分群揭示三類球迷

為刻劃偏好的個體異質性，本研究並行兩條分群路徑。其一以受訪者的選擇行為特徵（價格敏感度、皆不選比例、方向性屬性勝率）執行 K 平均集群分析（K-means）；其二以球隊認同、球迷社群認同與知覺價值三組共十九題李克特量表執行潛在類別分析（latent class analysis；Latent Class）。兩條路徑均報告三群結果，以兼顧資料驅動的最適解與群數的可解釋性（潛在類別與集群法綜述）。

行為軸區分價格敏感、內容黏著與票券觀望三型

行為分群（K 平均，Silhouette $=0.24$）將 74 位球迷分為三型，命名與規模整理如表 4。價格敏感型（n = 39，53%）的價格敏感度最高（看到較便宜方案即傾向選取）、皆不選比例低；內容黏著型（n = 28，38%）價格敏感度低、幾乎不選皆不選，顯示其入會決策由權益內容而非價格主導；票券觀望型（n = 7，9%）的皆不選比例最高、對免費票券的勝率最高，反映其對方案整體興趣有限、僅對票券誘因有反應。

態度軸區分三個認同強度層級

態度潛在類別分析將球迷依球隊、社群與價值認同強度分為三個層級：中度認同球迷（n = 26，三量表均值約 4.6 至 4.9）、高度認同球迷（n = 27，約 5.4 至 5.9）、與鐵粉等級的極高認同球迷（n = 21，約 6.0 至 6.2）。兩條分群路徑的交叉表 Cramér's V 僅為 0.12（$p \approx .70$），顯示球迷的「選擇行為型態」與「態度認同強度」近乎獨立——亦即高認同球迷未必對價格不敏感，價格敏感者亦未必認同度低。兩軸因而並列呈現，分別刻劃行為區隔與態度潛在類別。

模型配適與信效度檢驗支持主結論

模型在保留任務上的樣本外命中率為 56.1%（高於三選一隨機基準 33.3%），保留集 McFadden pseudo-$R^2$ 為 0.16；以一百次任務軸折半重抽估得的 Spearman-Brown 信度，屬性重要性為 .82、部分效用為 .84；三組態度量表的 Cronbach's $\alpha$ 介於 .85 至 .90；共同方法變異標記題與知覺價值的相關接近零（$r = -.01$），顯示共同方法偏誤不嚴重；資料飽和曲線於現有樣本已大致平穩。混合羅吉特（隨機參數）對條件羅吉特的概似比檢定顯著（$\mathrm{LR} = 44.65$, $\mathit{df} = 16$, $p < .001$），表示存在個體偏好異質性、IIA 假設不足，惟兩模型的係數同號全數通過（16/16），且個體層次異質性已由潛在類別分群吸收，故主分析所揭露的偏好方向穩健（IIA 處理與完整檢驗結果詳見附錄 A）。綜合各項指標，命中率、信度與量表效度均達或接近慣例門檻，僅統計考驗力受限於樣本規模而偏低（整體約 .38，將於限制一節討論），整體信效度尚足以支撐前述偏好結構與分群之推論。
