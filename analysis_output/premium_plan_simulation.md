# 最高價模擬會員方案（成熟購票球迷子樣本 WTP）

> WTP 來源：成熟購票球迷子樣本（排除 ≤24 歲與無購票經驗者，有效 n = 87）。
> 設計原則：各屬性挑願付金額最高的水準堆疊；互動活動全水準 WTP 為負（球迷反感），
> 故方案不納入任何互動活動（設「無互動」）。

## 方案配置與各屬性貢獻 WTP
| 屬性 | 選定水準 | 相對最低水準之 WTP (元/年) |
| --- | --- | ---: |
| 優先購票順位 | 第二順位（每場 4 張） | $+1,706$  |
| 免費票券 | 免費票券 4 張 | $+1,254$  |
| 獨家媒體內容 | 球員深度專訪 | $+672$  |
| 會員周邊 | 會員專屬 T 恤 | $+273$  |
| 互動活動 | 無互動活動 | $+0$ （不納入；互動 WTP 為負） |

## 堆疊願付金額（方案總價值上限）
- 點估計：**3,895 元/年**（Krinsky-Robb 95% CI：$[2,494, 7,017]$，5000 次抽樣）
- 此為「若同時提供第二順位購票、4 張免費票、球員深度專訪、專屬 T 恤」相對於最陽春方案（第四順位、無票券、賽後戰報、無贈品、無互動）的總願付溢價。

## 候選定價與消費者淨值
> 淨值 = 堆疊 WTP − 定價；為正表示對成熟球迷仍「物有所值」。
| 定價（元/年）| 消費者淨值（點估計）| 相對堆疊 WTP 上限 |
| ---: | ---: | :---: |
| 2,200 | $+1,695$ | 56% ✅ 仍具吸引力 |
| 2,500 | $+1,395$ | 64% ✅ 仍具吸引力 |
| 2,900 | $+995$ | 74% ✅ 仍具吸引力 |
| 3,200 | $+695$ | 82% ✅ 仍具吸引力 |

## 定價建議
- 堆疊 WTP 上限約 3,895 元，95% CI 下界 2,494 元。
- 保守定價（抓 CI 下界的安全邊際）：建議不超過 **2,494 元/年**，以確保多數成熟球迷仍感物有所值。
- 若以點估計 8 折定價（約 3,116 元），可留給消費者明顯淨值、提升轉換。
- 互動活動不納入高價方案：球迷對抽選制見面會反感，加入反而壓低方案價值與轉換率；若要做互動，宜另案處理或改設計（非抽選、明確中籤）。

## 方法說明與限制（堆疊 WTP 的正當性與邊界）

- **為何可相加**：在條件羅吉特的線性可加效用設定下，方案總效用為各屬性部分效用之和，且每項 WTP 皆以同一個 $|\beta_{price}|$ 為分母，故各屬性 WTP 可直接加總，等價於「滿配方案相對陽春方案的總效用差 ÷ $|\beta_{price}|$」。此為 conjoint 定價模擬的標準做法。
- **限制一（可加性）**：堆疊假設屬性間無交互作用、效用嚴格線性可加。惟本研究的 IIA 檢驗已檢出顯著偏好異質性、且球隊間存在交互效果，故堆疊 WTP 為「平均球迷」的一階近似，非每位受訪者的真實加總。
- **限制二（可能高估套裝價值）**：分項估值加總常高於消費者對「整包方案」的實際評價（套裝折價／邊際效用遞減），故 3,895 元宜視為**定價上限**而非目標售價。
- **限制三（CI 偏寬）**：WTP 為兩參數之比，分母 $\beta_{price}$ 在小樣本（n=87）下估計不穩會使比值厚尾，故 95% CI 上界（7,017 元）偏高、參考價值有限；定價應以點估計與 CI 下界為準。
- 實際定價尚須考量成本、競品（見 competitor_benchmark）與市場滲透目標。
