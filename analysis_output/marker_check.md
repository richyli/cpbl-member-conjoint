# CMV Marker 檢驗（marker 題：「我很喜歡白色」）

> marker 為理論上與會員偏好無關的標記變項（Lindell & Whitney, 2001）。
> 若 marker 與實質構念/偏好相關偏低，表示共同方法變異（CMV）不嚴重。

- 取 marker 與量表的最小絕對相關作 CMV 估計：$r_M = 0.075$
- 校正前後量表間相關（部分相關 $r_A = (r - r_M)/(1 - r_M)$）：

## 判定
- marker × 量表最大 |r| = 0.102；marker × 屬性勝率最大 |r| = 0.089
- 校正前後量表相關最大變動 = 0.022
- ✅ **CMV 不嚴重**：marker 與實質構念、與 conjoint 偏好相關皆低（|r| < .20），且 Lindell-Whitney 校正不改變量表相關結構，共同方法偏誤未構成威脅。
